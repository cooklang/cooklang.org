CookCLI

The Cook CLI command line program provides a suite of tools to create shopping lists, maintain recipes, and manage your cooking workflow. We've built it to be simple and useful for automating your cooking and shopping routine with existing UNIX command line and scripting tools.

Available Commands

Click on any command below to see detailed documentation:

  • recipe – Parse and display recipe files in various formats
  • shopping-list – Generate shopping lists from multiple recipes
  • server – Run a web server to browse your recipe collection
  • search – Search through your recipes by ingredient or text
  • import – Import recipes from websites and convert to Cooklang
  • doctor – Validate recipes and check for issues
  • seed – Initialize a directory with example recipes
  • report – Generate custom reports using templates
