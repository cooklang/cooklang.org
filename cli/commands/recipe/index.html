Recipe

The recipe command parses and displays Cooklang recipe files. It's your primary tool for viewing recipes, validating syntax, and converting between formats.

Basic Usage

cook recipe &#34;Neapolitan Pizza.cook&#34;

File extension is optional, that works too:

cook recipe &#34;Neapolitan Pizza&#34;

This displays the recipe in a human-readable format with ingredients, steps, and metadata clearly organized.

Menu Files

CookCLI also supports .menu files for meal planning. Menu files can reference multiple recipes and organize them by meals or days:

# View a menu file
cook recipe &#34;2 Day Plan.menu&#34;

# Scale entire menu (scales all referenced recipes)
cook recipe &#34;2 Day Plan.menu:2&#34;

Menu files use the same scaling notation as regular recipes, and the scaling applies to all recipe references within the menu.

Reading Recipes

The simplest way to view a recipe:

# View a recipe file
cook recipe &#34;Pasta Carbonara.cook&#34;

# Read from stdin
cat recipe.cook | cook recipe

# View with explicit path
cook recipe ./recipes/italian/Pizza.cook

Scaling Recipes

Scale recipes on the fly using the : notation or the --scale flag:

# Double a recipe
cook recipe &#34;Pizza.cook:2&#34;

# Scale to 1.5x
cook recipe &#34;Cake.cook&#34; --scale 1.5

# Quarter a recipe
cook recipe &#34;Soup.cook:0.25&#34;

All ingredient quantities are automatically adjusted:

Original:                    Scaled x2:
flour         500 g    →     flour         1 kg
water         300 ml   →     water         600 ml

JSON Format

Perfect for processing with other tools:

cook recipe &#34;Neapolitan Pizza&#34; -f json | jq &#39;.ingredients&#39;

Output:

[
  {
    &#34;name&#34;: &#34;semolina&#34;,

      &#34;reference_target&#34;: null
    },
    &#34;modifiers&#34;: &#34;&#34;

YAML Format

cook recipe &#34;Neapolitan Pizza&#34; -f yaml

Markdown Format

Great for documentation or sharing:

cook recipe &#34;Neapolitan Pizza&#34; -f markdown &gt; recipe.md

Cooklang Format

Regenerate clean Cooklang markup:

cook recipe &#34;Neapolitan Pizza&#34; -f cooklang

LaTeX Format

Export recipes as LaTeX documents for professional typesetting:

cook recipe &#34;Neapolitan Pizza&#34; -f latex

# Pipe directly to pdflatex to create a PDF
cook recipe &#34;Neapolitan Pizza&#34; -f latex | pdflatex -jobname=&#34;pizza-recipe&#34;

Schema.org Format

Generate structured data in Schema.org Recipe format for SEO and web integration:

cook recipe &#34;Neapolitan Pizza&#34; -f schema

Saving Output

Write the output to a file:

# Save as JSON
cook recipe &#34;Neapolitan Pizza&#34; -f json -o pizza.json

# Save as Markdown
cook recipe &#34;Neapolitan Pizza&#34; -f markdown -o pizza.md

# Format is inferred from extension
cook recipe &#34;Neapolitan Pizza&#34; -o pizza.yaml

Pretty Printing

For JSON and YAML outputs, use --pretty for formatted output:

cook recipe &#34;Neapolitan Pizza&#34; -f json --pretty

Recipe Discovery

CookCLI can find recipes by name without the full path:

# Searches in current directory and subdirectories
cook recipe &#34;Pizza&#34;

# Automatically adds .cook extension
cook recipe &#34;Pasta Carbonara&#34;

# Searches in recipe directories
cook recipe &#34;Neapolitan Pizza&#34;

Advanced Examples

Recipe Analysis Pipeline

Combine with UNIX tools for analysis:

# Count total ingredients across all recipes
for recipe in *.cook; do
  cook recipe &#34;$recipe&#34; -f json
done | jq -s &#39;[.[].ingredients[].name] | group_by(.) | map({name: .[0], count: length})&#39;

Batch Processing

Process multiple recipes:

# Convert all recipes to Markdown
for recipe in *.cook; do
  name=$(basename &#34;$recipe&#34; .cook)
  cook recipe &#34;$recipe&#34; -f markdown -o &#34;docs/${name}.md&#34;
done

# Validate all recipes
for recipe in *.cook; do
  echo &#34;Checking $recipe...&#34;
  cook -vvv recipe &#34;$recipe&#34; &gt; /dev/null || echo &#34;Error in $recipe&#34;
done

Recipe Comparison

Compare scaled versions:

# Compare original and doubled recipe
diff &lt;(cook recipe &#34;Neapolitan Pizza&#34;) &lt;(cook recipe &#34;Neapolitan Pizza:2&#34;)
