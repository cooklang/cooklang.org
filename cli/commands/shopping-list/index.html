Shopping List

The shopping-list command creates organized shopping lists from one or more recipes. It automatically combines ingredients, converts units (some day!), and groups items by store section.

Basic Usage

cook shopping-list &#34;Pasta.cook&#34; &#34;Salad.cook&#34;

This creates a combined shopping list with all ingredients from both recipes.

Creating Shopping Lists

Single Recipe

cook shopping-list &#34;Breakfast/Easy Pancakes&#34; &#34;Neapolitan Pizza&#34;

Output:

[dried herbs and spices]
salt                     24.6 g
sea salt                 1 pinch

[milk and dairy]
eggs                     3

[oils and dressings]
oil
olive oil                1 drizzle

[other]
San Marzano tomato sauce 5 tbsp
basil leaves
mozzarella cheese        100 grams
semolina

Multiple Recipes

Combine ingredients from several recipes:

cook shopping-list &#34;Pizza.cook&#34; &#34;Caesar Salad.cook&#34; &#34;Tiramisu.cook&#34;

# Using wildcards
cook shopping-list *.cook

# Specific pattern
cook shopping-list Dinner/*.cook Desserts/*.cook

Scaling Recipes

Scale individual recipes in your shopping list:

# Double the pizza, regular salad
cook shopping-list &#34;Pizza.cook:2&#34; &#34;Salad.cook&#34;

# Different scaling for each
cook shopping-list &#34;Pasta.cook:3&#34; &#34;Bread.cook:0.5&#34; &#34;Soup.cook:2&#34;

# Dinner party for 8
cook shopping-list &#34;Main Course.cook:2&#34; &#34;Side Dish.cook:2&#34; &#34;Dessert.cook:2&#34;

Menu Files

Create shopping lists from .menu files that organize multiple recipes:

# Generate shopping list from weekly menu
cook shopping-list &#34;2 Day Plan.menu&#34;

# Scale entire menu for more people
cook shopping-list &#34;2 Day Plan.menu:2&#34;

# Combine menu with individual recipes
cook shopping-list &#34;2 Day Plan.menu&#34; &#34;Extra Snacks.cook&#34;

Menu files can contain recipe references with their own scaling, and the menu-level scaling multiplies with individual recipe scales.

Output Formats

Human-Readable (Default)

Organized by store section:

cook shopping-list &#34;Pizza.cook&#34; &#34;Pasta.cook&#34;

Plain List

Without categories:

cook shopping-list &#34;Pizza.cook&#34; --plain

Output:

flour                         1 kg
tomatoes                      500 g
mozzarella                    200 g
olive oil                     50 ml

JSON Format

For integration with other tools:

cook shopping-list &#34;Pizza.cook&#34; -f json
{
  &#34;categories&#34;: {
    &#34;produce&#34;: [
      {&#34;name&#34;: &#34;tomatoes&#34;, &#34;quantity&#34;: &#34;500 g&#34;}
    ],
    &#34;dairy&#34;: [
      {&#34;name&#34;: &#34;mozzarella&#34;, &#34;quantity&#34;: &#34;200 g&#34;}
    ]
  }
}

YAML Format

cook shopping-list &#34;Pizza.cook&#34; -f yaml

Markdown Format

Perfect for sharing or printing:

cook shopping-list &#34;Menu/*.cook&#34; -f markdown &gt; shopping.md

Aisle Configuration

Organize items by store section using aisle.conf:

# Default location: ./config/aisle.conf
[produce]
tomatoes
onions
garlic
basil

[dairy]
milk
cheese
yogurt
mozzarella

[meat]
chicken
beef
pork

[pantry]
flour
sugar
salt
pasta

Using Custom Aisle Configuration

cook shopping-list &#34;Recipe.cook&#34; --aisle ~/my-store-layout.conf

Checking Missing Aisles

Find ingredients not in your aisle configuration:

cook doctor aisle

Pantry Configuration

Track your ingredient inventory and automatically exclude items you already have from shopping lists using pantry.conf:

# Default location: ./config/pantry.conf
[freezer]
ice_cream = &#34;1%L&#34;
frozen_peas = &#34;500%g&#34;
spinach = { bought = &#34;05.05.2024&#34;, expire = &#34;05.06.2024&#34;, quantity = &#34;1%kg&#34; }

[fridge]
milk = { expire = &#34;10.05.2024&#34;, quantity = &#34;2%L&#34; }
cheese = { expire = &#34;15.05.2024&#34; }
butter = &#34;250%g&#34;

[pantry]
rice = &#34;5%kg&#34;
pasta = &#34;1%kg&#34;
flour = &#34;5%kg&#34;
salt = &#34;1%kg&#34;
olive_oil = &#34;1%L&#34;

How It Works

Items listed in your pantry are automatically excluded from shopping lists:

# Recipe calls for: flour, eggs, milk, sugar
# Pantry has: flour (5kg), milk (2L)
# Shopping list shows only: eggs, sugar
cook shopping-list &#34;Cake.cook&#34;

Enable logging to see what's excluded with pantry:

cook -v shopping-list &#34;Cake.cook&#34;

Pantry Item Formats

Two ways to specify items:

  1. Simple format: item = "quantity"

    rice = &#34;5%kg&#34;

  2. Detailed format: Track expiration and purchase dates

    milk = { quantity = &#34;2%L&#34;, expire = &#34;10.05.2024&#34;, bought = &#34;05.05.2024&#34; }

Using Custom Pantry Configuration

cook shopping-list &#34;Recipe.cook&#34; --pantry ~/my-pantry.conf

Recipe References

Shopping lists handle recipe references (includes) automatically:

# Main.cook
Include @./Pizza Dough{}

Add @tomato sauce{200%ml} and @mozzarella{150%g}.
# Includes ingredients from both Main.cook and Pizza Dough.cook
cook shopping-list &#34;Main.cook&#34;

To ignore references:

cook shopping-list &#34;Main.cook&#34; --ignore-references

Saving Lists

To File

# Save as text
cook shopping-list Menu/*.cook -o list.txt

# Save as JSON
cook shopping-list Menu/*.cook -f json -o list.json

# Format inferred from extension
cook shopping-list Menu/*.cook -o shopping.yaml

Ingredients Only

Just list ingredient names without quantities:

cook shopping-list &#34;Recipe.cook&#34; --ingredients-only

Output:

flour
eggs
milk
sugar

Advanced Usage

Weekly Meal Planning

Create a shopping list for the week:

# Week&#39;s menu
cook shopping-list \
  Monday/Pasta.cook:2 \
  Tuesday/Stir-fry.cook:2 \
  Wednesday/Pizza.cook:3 \
  Thursday/Soup.cook:4 \
  Friday/Fish.cook:2 \
  -o weekly-shopping.txt

Party Planning

Scale recipes for large gatherings:

# Party for 20 people (recipes serve 4)
scale=5
cook shopping-list \
  &#34;Appetizer.cook:$scale&#34; \
  &#34;Main Course.cook:$scale&#34; \
  &#34;Side Dish.cook:$scale&#34; \
  &#34;Dessert.cook:$scale&#34;

Smart Shopping

Batch Cooking

Plan bulk preparation:

# Make 10 portions for freezing
cook shopping-list &#34;Freezer Meals/*.cook:10&#34; \
  -o bulk-cooking-list.txt

Common Issues

Missing Aisle Categories

Items not in aisle.conf appear in an "Other" category. Run cook doctor aisle to find and fix these.

Recipe Not Found

Use full paths or ensure recipes are in the current directory:

# Explicit path
cook shopping-list ~/recipes/Pizza.cook

# Change directory first
cd ~/recipes &amp;&amp; cook shopping-list Pizza.cook
